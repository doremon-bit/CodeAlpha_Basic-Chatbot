# CodeAlpha Basic Chatbot

## Description
A simple rule-based chatbot developed using Python. The chatbot interacts with users through predefined responses and continues the conversation until the user enters "bye".

## Features
- Greets the user
- Responds to basic questions
- Exits when the user types "bye"
- Handles unknown inputs

## Concepts Used
- Python Functions
- While Loop
- If-Else Statements
- User Input and Output
- String Methods

## How to Run

1. Make sure Python is installed.
2. Run the following command:

```bash
python chatbot.py
```

## Sample Commands

- hello
- how are you
- bye
